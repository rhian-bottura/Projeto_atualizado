O que é o projeto?

O View é uma plataforma voltada para a aquisição de dados de iluminação das ruas da cidade de São Paulo para que possam ser consultados a fim de informar motoristas quais trechos estão pouco iluminados.

Segurança

Proporciona mais segurança para os motoristas de aplicativo e usuários de apps de rotas

IOT

Tecnologia IOT como forma de garantia da precisão das informações

Velocidade

Nosso banco de dados é atualizado em tempo real com os dados captados pelos sensores que estarão circulando por SP

Intregantes

A equipe de desenvolvimento do projeto é constituida
por alunos da faculdade Bandtec, que cursam Ciência da computação
